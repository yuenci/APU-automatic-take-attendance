## Getting Started

Enter the following pip command in cmd to install the dependency

### Prerequisites

```bash
$ pip install -r requirements.txt
```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/yuenci/APU-automatic-take-attendance
   ```
2. Run main.py
